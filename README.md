# doh-proxy

Deploy this github repository to [Cloudflare Pages](https://pages.cloudflare.com/), and you will get a reversed doh service(DOH3 Supported).

Cloudflare:
```
https://{your_domain}/cloudflare
```

```
https://{your_domain}/google
```
